What is Convolution?

Convolution is a mathematical operation where filters scan images to detect patterns such as edges, textures, and shapes.

Why Pooling is Used?

Pooling reduces image size and helps the model focus on important features while reducing computation.

Why ReLU is Used?

ReLU introduces non-linearity and helps the neural network learn complex patterns efficiently.
